Lubuntu/Linux 新手一鍵安裝指南(64位元方案)

  • 特 色
    ① 安裝快速
    ② 一鍵懶人設定檔自動布屬
    ③ vim/tmux/桌面快鍵 含好用的基本設定

  • 安 裝
    ① 安裝虛擬機virtual box32

    ② 用virtual box安裝Lubuntu x64 or Lubuntu x32
    一直next,不用選（只有一步要調成繼續）
    （安裝完成要Restart的時候,直接把虛擬機整個關閉再重開,不然會卡住）

    ③ 密碼登入桌面

  • 設 定
    ④ 桌面左下角點開browser (火狐)
    輸入網址：http://yale918.github.io/newhand/

    ⑤ 全自動設定檔下載
    [ save file ] => [ ok ]

    ⑥ Ctrl-Alt + t 打開Terminal
    輸入： . ~/Downloads/autoInstall.sh
    他會自動跑設定,大約5分鐘

  • 安裝完後調整
    ① 無縫模式: [裝置]> [guest addition] > 重新開機> [檢視]
    ② 雙向剪貼模式: [裝置]>[共用剪貼簿]
     ③ ibus添加新酷音:[開始]> [Preferences]>[Language Support]>keyboard input system : 改成iBus
    [Preferences]>[iBus Preferences]>[input method]> add 新酷音
    重新開機
    ps.1 tmux的快速鍵設定: ~/.tmux.conf
    vim的設定: ~/.vimrc
    desktop快速鍵設定: ~/.config/openbox/Lubuntu-rc.xml
    alias在這邊設定: ~/bash_aliases
    ps.2 有無法執行的地方請跟我說,我再修正：）
